Politique de confidentialité - Code Scan

Date d'effet: 2026-01-05

Résumé

  • Pas de compte utilisateur.
  • Traitement local sur l'appareil pour le scan.
  • Données stockées localement: historique, favoris, notes, préférences.
  • Pas de serveur, pas de collecte publicitaire.
  • Partage uniquement si vous l'initiez (partage Android ou envoi vers PC via Bluetooth).

Données traitées

  • Caméra: le flux vidéo est utilisé pour détecter les codes-barres. Les images ne sont pas stockées ni envoyées.
  • Historique de scan: codes, type, date/heure, notes, favoris.
  • Bluetooth (PC Link): nom de l'appareil, statut de connexion; les codes sont envoyés au PC comme saisie clavier.
  • Presse-papiers: copie du code lorsque vous utilisez la fonction de copie.

Utilisation des données

  • Scanner des codes-barres et afficher le résultat.
  • Gérer l'historique, les favoris et les notes.
  • Envoyer un code vers un PC via Bluetooth (mode douchette).

Partage et services tiers

  • Aucune vente de données et pas de publicité.
  • Pas de serveur d'application: les données restent sur l'appareil.
  • Bibliothèque ML Kit (Google) utilisée pour la lecture des codes-barres, avec traitement local sur l'appareil.
  • Le partage ne se fait que si vous l'initiez (intent Android, envoi Bluetooth).

Stockage et sécurité

Les données sont stockées localement sur l'appareil. Vous pouvez supprimer l'historique depuis l'application. Si la sauvegarde Android est activée, certaines données peuvent être sauvegardées par le système.

Permissions

  • Caméra: nécessaire pour scanner.
  • Bluetooth: nécessaire pour le mode PC Link.
  • Vibration: retour haptique lors de la validation d'un scan.

Enfants

L'application n'est pas conçue pour les enfants de moins de 13 ans et ne collecte pas sciemment de données d'enfants.
