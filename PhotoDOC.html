Règles de Confidentialité pour Photo DOC IA

Dernière mise à jour : 24 septembre 2025

Introduction

Bienvenue sur Photo DOC IA. Ces règles de confidentialité expliquent comment INJALRAC Frédéric ("nous", "notre") collecte, utilise et protège vos informations lorsque vous utilisez notre application mobile Photo DOC IA (l'"Application").

La confiance de nos utilisateurs est notre priorité. Nous avons conçu l'Application pour qu'elle soit la plus respectueuse possible de votre vie privée.

Informations que nous collectons

L'Application traite les types d'informations suivants :

  • Images et Photos : Les images que vous capturez avec l'appareil photo ou que vous sélectionnez depuis la galerie de votre appareil. Ces images sont traitées localement sur votre appareil.
  • Texte extrait des images : Le texte qui est généré par le processus de reconnaissance optique de caractères (OCR) à partir de vos images. Ce traitement est également effectué localement sur votre appareil.
  • Clés d'API personnelles : Les clés d'API que vous fournissez pour les services d'intelligence artificielle tiers (OpenAI, Google Gemini, Anthropic).

Nous ne collectons aucune information personnelle identifiable automatiquement (comme votre nom, adresse e-mail, ou localisation) sans votre action explicite.

Comment nous utilisons vos informations

Les informations que vous fournissez sont utilisées exclusivement pour les fonctionnalités principales de l'Application :

  • Traitement de documents : Vos images sont utilisées pour extraire du texte (OCR) et segmenter des images. Ce processus se déroule entièrement sur votre appareil.
  • Amélioration par IA : Le texte extrait de vos documents est envoyé au service d'IA que vous avez choisi et configuré dans les paramètres (OpenAI, Google Gemini, ou Anthropic) afin d'être corrigé et amélioré.
  • Authentification aux services tiers : Vos clés d'API sont stockées de manière sécurisée dans un fichier privé de l'application sur votre appareil. Elles sont utilisées uniquement pour authentifier vos requêtes auprès du service d'IA correspondant. Elles ne sont jamais partagées avec nous ou avec d'autres services.
  • Sauvegarde sur Google Drive (fonctionnalité future) : Si vous choisissez de vous connecter à votre compte Google, l'application utilisera votre autorisation pour créer et sauvegarder les documents traités directement sur votre propre Google Drive / Google Docs.

Partage de vos informations

Nous ne vendons, n'échangeons, ni ne louons vos informations personnelles à des tiers. Le partage de vos données est limité aux cas suivants, qui sont sous votre contrôle direct :

  • Fournisseurs d'IA : Le texte de vos documents est partagé avec le service d'IA que vous avez sélectionné (OpenAI, Google Gemini, ou Anthropic) lorsque vous lancez un traitement. Nous vous invitons à consulter les politiques de confidentialité de ces services pour comprendre comment ils traitent vos données.
  • Google Drive / Google Docs : Lorsque vous utilisez la fonction d'export, le document final est envoyé aux serveurs de Google pour être sauvegardé sur votre compte.

Sécurité des données

La sécurité de vos données est primordiale.

  • Toutes les clés d'API et les préférences sont stockées dans un fichier privé sur votre appareil via le composant sécurisé Jetpack DataStore. Aucune autre application ne peut y accéder.
  • Toute communication avec les services d'IA externes et Google se fait via une connexion sécurisée et chiffrée (HTTPS).

Conservation des données

L'Application ne conserve pas vos images ou le contenu de vos documents une fois le traitement terminé. Les fichiers temporaires peuvent être stockés dans le cache de l'application et sont supprimés par le système d'exploitation au besoin. Les préférences et les clés d'API que vous avez saisies restent sur votre appareil jusqu'à ce que vous les modifiiez, les supprimiez ou que vous désinstalliez l'application.

Modifications de cette politique de confidentialité

Nous pouvons mettre à jour nos règles de confidentialité de temps à autre. Nous vous notifierons de tout changement en publiant les nouvelles règles sur cette page. Il est conseillé de consulter cette page périodiquement pour tout changement.
